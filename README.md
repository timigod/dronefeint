# DroneFeint

A web-based game with an infinite looping digital map.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open your browser to the URL shown (usually http://localhost:5173)

## Controls

- **Click and drag**: Pan the map
- **Mouse wheel**: Scroll the map
- The map loops infinitely in all directions!

## Project Structure

- `src/game/Map.tsx` - Main game map component with canvas rendering
- `src/game/types.ts` - TypeScript type definitions
- `src/main.tsx` - Application entry point
